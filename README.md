# HelloWorld

LLM ドリブンな iOS アプリ開発そのものを目的にした実験プロジェクトである。

## Overview

このアプリは公開を目指すものではなく、LLM と人が協調して設計、実装、検証、文書更新を回せるかを試すプロジェクトである。
機能としては、YouTube チャンネルの登録、動画一覧の閲覧、検索結果の確認、ローカルキャッシュ運用、簡易バックアップを備えた iOS アプリである。
