# Session Start Rules

この文書は、開発セッション開始という 1 つのタスクだけを定義する完結文書である。

## タスク定義

- セッション開始とは、作業開始前の基準状態を整えるタスクである。
- セッション開始タスクは、次の 2 条件を満たした時だけ完了とみなす。

## 完了条件

- Git の現在ブランチが `main` であり、その `main` が最新であること。
- すべての `*-latest.md` 文書について、次を保証すること。
  - 本日以外の日付見出しを含む内容は、対応する `*-log.md` へ掃き出されていること。
  - `*-latest.md` の 1 行目が本日の日付見出しであること。

## 禁止事項

- 上記 2 条件以外の作業を、この文書の対象へ含めてはならない。
- セッション開始タスクが未完了のまま、以後の開発タスクへ進んではならない。
